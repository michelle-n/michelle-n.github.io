Project Details of dpfplusplus

Distributed point functions (DPFs) are a cryptographic primitive, used for secret sharing, featured in many recent anonymous messaging systems: such as Riposte (Corrigan-Gibbs et al., 2015) or Sabre (Henry et al., 2021).

dpfplusplus is a C++ implementation of (2,2)-distributed point functions (see Gilboa and Ishai, 2014) by Dr. Ryan Henry and Dr. Adithya Vadapalli.

My work on this project originated from a senior-level seminar course taken at the University of Calgary with Dr. Ryan Henry. In this project, my colleague (Nickole Jimenez) and I wrote documentation and performed maintenance on the codebase. As well, I attempted to optimize functions within the codebase by modifying the access patterns and data structures used, to allow for better caching; however, this code would ultimately not make it to production as the algorithm/underlying mathematics attempted still need to be resolved. Below are some of my key contributions to the project.

  • Increasing user-friendliness for new users, by creating the first tutorials on getting started with dpfplusplus. Three tutorials and a template were created, which included step-by-step walkthroughs and minimal code examples.
  • Simplifying user-experience, by establishing a documentation website for dpfplusplus and implementing a design for quick access to code/function-specific documentation and tutorials.
