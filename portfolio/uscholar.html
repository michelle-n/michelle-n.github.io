Project Details of uScholar

uScholar is a (mock) academic journal submission web application, created for a software engineering course at the University of Calgary. The web application was created with my colleagues, Misael Esperanzate, Andrew Truong, and Jonathan Delos Santos, from February 2020 to April 2020. Mid-way through this period, the Covid-19 lockdown began in Calgary and our team had to adapt, improvise, and overcome the challenges of suddenly working remotely. Following Agile practices and quickly acquainting ourselves with online meeting/communication/collaborating software, we were able to complete all the features dictated by our user stories to produce a working journal submission web application.

Some of the user stories I worked on developing were:

  • As an editor, I want to access all submissions so I can view their progress.
  • As an editor, I want to view the submitted paper assigned to me after the deadline so I can accept or reject them for publication.
  • As an editor, I want to set deadlines for the reviewers to submit their feedback.

Features to address these stories were implemented in the "editor's" route/view of the web application; screenshots of this view are shown in the above slideshow.

Overall, my achievements were:

  • Collaborating with a team to tackle a web application for academic journal submissions, despite the sudden challenges brought by Covid-19.
  • Improving database efficiency by designing the structure to reduce nesting and flatten data structures.
