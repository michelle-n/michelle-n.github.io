Project Details of Jumpy Man

Jumpy Man is a platformer game where the player must adventure through a series of levels against time. On each level, the player is faced with enemies which they must either avoid or defeat using their firing weapon. But itâ€™s not just about finishing the levels quickly-- the player can increase their score by trying to collect all the coins along the way!

I created this game with my partners: Gao XingYu and John Hutchinson. We created this project over the course of 6-weeks while learning Java. My key contributions to this project were:

  • Creating a playable story: by developing the logic for moving between levels, and states detecting winning or losing, timing and scoring.
  • Enhancing gameplay experience: by implementing the GUI corresponding to objects (platforms, player, enemies, coins); developing the game camera (moving the background as the character progresses); and designing the game theme, used in-game and in-game menus.
  • Making fun challenges for players: by designing two game levels, each of which incorporate random-moving enemies and platform 'puzzles' to test the player's jumping skills.
